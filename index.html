Group Project 1 — Travel: Journey to Common Ground

Overview + my contributions

Project overview

Four travelers journey to the same destination. The destination starts incomplete and monochrome. Each traveler has a “lens” (button) that reveals part of the world. Only when all perspectives activate does “Common Ground” appear in full color.

Goal: make teamwork and shared contribution obvious at a glance.

My individual contributions

  • Divergence: Designed and sketched the Phone Home Screen concept + Coffee Diary interaction.
  • Convergence / assets: Edited the destination image in Photoshop (overlay + grayscale base + layered elements).
  • Implementation: Coded the scene transition from monochrome → full color (button-triggered reveal logic + UI layout).
  • Documentation: Wrote/compiled major sections of the GP1 report and coordinated revisions.

Reflection (what I learned)

In early ideation, I leaned toward familiar UI metaphors (like a phone home screen) because they feel intuitive. However, critique made it clear that familiar metaphors can accidentally encourage isolated interaction—each “app” becomes separate rather than collaborative.

This pushed my thinking toward designs where collaboration is not optional: individual actions should visibly accumulate and create a shared outcome. Going forward, I want to prototype interactions that make interdependence clear without requiring explanation.
